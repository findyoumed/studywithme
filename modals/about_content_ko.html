StudyWithMe 소개

Video Study Trainer (StudyWithMe)은 유튜브 스터디 영상을 보며 집에서 집중 학습할 수 있도록 도와주는 AI 학습 코칭 서비스입니다.

여러분의 피드백으로 StudyWithMe를 더 좋게 만들 수 있어요.

주요 기능

  • AI 학습 집중도 분석: 실시간으로 움직임을 분석하여 학습 집중도를 점수로 제공합니다.
  • 유튜브 연동: 원하는 스터디 영상을 검색하거나 URL로 추가해 따라할 수 있습니다.
  • 프라이버시 보호: 영상 데이터는 브라우저 내에서만 처리되며 서버로 전송되지 않습니다.
  • 플레이리스트 관리: 자주 보는 스터디 영상을 저장하고 관리할 수 있습니다.
  • 환경 최적화: 화면 투명도, 민감도, 재생 속도 등을 설정에서 자유롭게 조절할 수 있습니다.

사용 방법

  1. 영상 선택: 플레이리스트에서 스터디 영상을 선택하거나 YouTube에서 검색하세요.
  2. 카메라 권한 허용: 학습 집중도 분석을 위해 카메라 권한을 허용해주세요.
  3. 공부 시작: 영상과 함께 공부하면 AI가 실시간 SCORE를 알려줍니다.
  4. 설정 조절: 투명도, 민감도, 재생 속도를 조절해 학습 환경을 최적화하세요.

자주 묻는 질문

Q: 회원가입이 필요한가요?

A: 아니요. StudyWithMe는 별도의 회원가입 없이 바로 사용할 수 있습니다. 모든 데이터는 브라우저에 저장됩니다.

Q: 카메라 권한을 거부했어요. 어떻게 해야 하나요?

A: 브라우저 주소창 옆 카메라 아이콘을 클릭한 뒤 “허용”으로 변경하고 페이지를 새로고침하세요.

Q: YouTube API 키가 필요한가요?

A: 아닙니다. 웹 버전은 검색 기능을 자동으로 제공합니다. 검색이 되지 않는다면 일시적인 네트워크 문제일 수 있으니 잠시 후 다시 시도해주세요.

Q: 어떤 스터디 영상을 추천하나요?

A: 상체와 책상이 잘 보이는 스터디 영상을 추천합니다. 타이머, 화이트노이즈, 조용한 공부 환경이 포함된 영상이 좋습니다.

Q: 모바일에서도 사용할 수 있나요?

A: 네, 모바일 브라우저에서도 정상 동작합니다. 다만 화면이 넓은 PC 환경을 권장합니다.

Q: 점수는 어떻게 계산되나요?

A: AI가 카메라를 통해 사용자의 움직임을 실시간으로 감지합니다. 바른 자세로 움직임을 최소화하고 집중할수록 SCORE가 올라갑니다.

Q: 개인정보는 안전한가요?

A: 모든 영상 처리는 브라우저 내에서만 이루어지며 서버로 전송되지 않습니다. 개인정보는 안전하게 보호됩니다.

Q: 무료인가요?

A: 네, StudyWithMe는 완전 무료 서비스이며 광고 수익으로 운영됩니다.

Q: 오프라인에서도 사용할 수 있나요?

A: YouTube 영상을 재생하려면 인터넷 연결이 필요합니다. AI 처리는 로컬에서 수행됩니다.

Q: 어떤 브라우저를 지원하나요?

A: Chrome, Edge, Safari 등 최신 브라우저를 지원합니다. Chrome 환경을 추천합니다.

효과적인 학습을 위한 가이드

🎥 카메라 설정

  • 위치: 상체와 책상이 화면에 잘 보이도록 카메라를 배치하세요.
  • 거리: 카메라에서 1~2미터 정도 떨어지는 것이 적당합니다.
  • 높이: 가슴 높이 또는 눈높이에 설치하면 인식이 가장 정확해집니다.
  • 조명: 밝은 조명 아래에서 학습하면 AI 인식률이 향상됩니다.

📌 학습 자세 팁

  • 정면 유지: 상체가 카메라를 향하도록 앉아주세요.
  • 천천히 적응: 처음에는 0.75배속으로 영상 흐름에 익숙해지세요.
  • 투명도 조절: 카메라 투명도를 조절해 영상 속의 주인공 자세와 자신의 자세를 비교해보세요.
  • 집중 유지: 실시간으로 올라가는 SCORE를 보며 움직임을 최소화하고 학습에 몰입하세요.

🎯 SCORE 올리는 방법

  • 민감도 조절: 민감도 슬라이더로 움직임 감지 강도를 조절하세요.
  • 얼굴 가시성: 얼굴과 상체가 잘 보이는 자세가 점수 측정에 가장 유리합니다.
  • 배경 정리: 단색 배경에서 학습하면 AI 인식률이 높아집니다.

📋 추천 학습 루틴

  • 초급: 25분 집중 + 5분 휴식 (포모도로 1세트)
  • 중급: 50분 집중 + 10분 휴식 + 20분 복습
  • 상급: 90분 집중 + 15분 휴식 + 30분 문제풀이

개인정보처리방침

1. 수집하는 정보

StudyWithMe는 다음 정보를 수집합니다:

  • 플레이리스트 데이터: 사용자가 저장한 스터디 영상 목록 (브라우저 localStorage)
  • 설정 정보: 투명도, 민감도, 재생 속도 등의 사용자 설정
  • 검색어: 사용자가 입력한 검색어

2. 카메라 영상 처리

중요: 카메라로 촬영된 영상은 서버로 전송되지 않습니다.

  • 모든 영상 처리는 브라우저 내에서만 수행됩니다.
  • AI 분석은 로컬 엔진에서 실행됩니다.
  • 영상 데이터는 외부에 저장되거나 전송되지 않습니다.

3. 데이터 저장

모든 데이터는 브라우저의 localStorage에만 저장됩니다:

  • 플레이리스트 정보
  • 사용자 설정 (투명도, 민감도 등)

브라우저 데이터를 삭제하면 모든 정보가 함께 삭제됩니다.

4. YouTube API 사용

YouTube 영상 검색 기능은 Google YouTube Data API v3를 사용합니다:

  • 검색 API 키는 서비스에서 안전하게 관리되며 사용자에게 노출되지 않습니다.
  • 검색어만 YouTube로 전송되며 개인정보는 전송되지 않습니다.

5. 쿠키 및 유사 기술

StudyWithMe는 쿠키를 직접 사용하지 않으며 localStorage만 사용합니다. 다만, 아래의 경우 쿠키가 사용될 수 있습니다:

  • Google AdSense 광고 표시 (광고 쿠키)
  • YouTube 영상 재생 (YouTube 쿠키)

6. 이용약관

서비스 이용: StudyWithMe는 무료로 제공되며 광고 수익으로 운영됩니다.

면책 조항: 장시간 학습에 따른 건강 문제에 대해 StudyWithMe는 책임지지 않습니다. 적절한 휴식과 바른 자세를 유지하세요.

저작권: YouTube 영상의 저작권은 각 영상 제작자에게 있으며, StudyWithMe는 영상 재생 플랫폼만 제공합니다.
