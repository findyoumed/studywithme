- ✅ 반투명 버튼

- ✅ 깔끔한 아이콘

**오른쪽 (켜짐 상태):**

- ✅ **빨간 그라데이션** - 즉시 알아볼 수 있음
- ✅ **펄스 애니메이션** - 살아있는 느낌
- ✅ **시청자 수 표시** - 3 viewers
- ✅ **글로우 효과** - 눈에 확 띄는 강조

### 구현 방법

```css
.live-btn-icon.active {
  background: linear-gradient(135deg, #ef4444, #dc2626);
  animation: pulse 2s infinite;
}

@keyframes pulse {
  0%,
  100% {
    box-shadow: 0 0 0 0 rgba(239, 68, 68, 0.7);
  }
  50% {
    box-shadow: 0 0 0 20px rgba(239, 68, 68, 0);
  }
}
```
